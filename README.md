# hjstagram

- React 컴포넌트 기반 UI 구축 및 Redux를 통한 상태관리
- 데이터 통신을 위한 Axios 사용과 Styled-components를 통한 스타일 정의
- Node.js와 Express로 RESTful API 서버 구축
- JWT를 사용한 사용자 인증 및 권한 부여를 처리 및 bcrypt를 통한 비밀번호 암호화 적용
